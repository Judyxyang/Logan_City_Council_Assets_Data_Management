A suite of three interactive tools built specifically for the WSUD Asset Management Officer role at LCC — demonstrating practical capability across whole-of-life asset management, stormwater quality compliance, offset frameworks and GIS-based spatial analysis.
Built against:  LCC ADAC Stormwater Standards · SEQ WSUD Design Guideline · LCC Stormwater Quality Offset Framework
Data:  170 synthetic WSUD asset records across 7 feature classes placed within Logan LGA
Grades:  1 = Excellent · 2 = Good · 3 = Fair · 4 = Poor · 5 = Critical (SEQ inspection scale)
